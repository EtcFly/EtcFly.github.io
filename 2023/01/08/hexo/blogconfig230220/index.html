下载Nodejs

  • 官网 下载地址

我这里系统是Ubuntu 18.04, 由于GLIBC版本比较低, 其他软件依赖等等原因, 这里只能安装1.6.x版本, 以上版本会报GLIBC 版本低。

下载文件目录是: ~/Tools

  • 添加环境变量
    echo "export PATH=~/Tools:$PATH" >> ~/.zshrc

  • 通过apt下载

检查版本

npm -v

如果输出版本
8.5.0

安装hexo

  linux下安装hexo如下:
npm install -g --save hexo-cli

安装完成可输入hexo -v查看版本,正常会显示类似如下结果:

  • 安装主题

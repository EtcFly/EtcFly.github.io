时间转换相关API

主要时间类型

  • struct tm： 描述当前的时分秒
  • struct timeval: 描述utc时间, 时间精度为usec毫秒
  • struct timespec:描述utc时间, 时间精度为nsec纳秒

系统主要时间函数就是围绕这几个类型展开

类型转换函数

  • 直接获取内核时间接口三个：

    • time: 获取当前秒
    • clock_gettime: 获取当前秒 + ns秒
    • gettimeofday: 获取当前秒 + ms秒 + 时区信息

  • time_t类型转为struct tm接口两个

    • gmtime: 转换的结果是0时区的日期, 需要自己加上时区信息
    • localtime： 转换的结果包含时区信息, 属于本地日期

  • struct tm 转为 time_t接口一个

    • mktime

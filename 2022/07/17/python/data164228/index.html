字典

字典(dictionary)是除列表以外 python 之中最灵活的内置数据结构类型。列表是有序的对象集合，字典是无序的对象集合。
两者之间的区别在于：字典当中的元素是通过键来存取的，而不是通过偏移存取。字典用"{ }"标识。字典由索引(key)和它对应的值value组成。字典的每个键值 key=>value 对用冒号 : 分割，每个键值对之间用逗号 , 分割

如:

keys:输出字典里所有的关键字 key 值。
values:输出字典里所有 value 值。
所有对象的访问都是通过key值来索引。相当于key/value关系数据库一样。

集合

集合（set）是一个无序的不重复元素序列, 可以使用大括号 { } 或者 set() 函数创建集合，注意：创建一个空集合必须用 set() 而不是 { }，因为 { } 是用来创建一个空字典。

集合的逻辑运算
  • &: 计算两个集合的交集
  • | : 计算两个集合的并集
  • ^: 计算两个集合公共部分意外的部分
  • -: 计算仅在集合a但是不存在于集合b的元素
集合的基本操作
  • add

为集合添加一个元素并返回新的集合对象

  • clear

移除集合的所有元素

  • copy

集合拷贝并返回一个新的集合

  • difference

返回两个集合的差集，生成新的对象

  • difference_update

移除原集合中两个集合的差集，不生成新的对象

  • discard

删除集合中指定元素

  • intersection

返回两个集合的交集, 生产新的对象

  • intersection_update

原结合移除两个集合不相交的元素, 不生产新的对象

  • isdisjoint

判断两个集合是否包含相同的元素，如果没有返回 True，否则返回 False。

  • issubset

判断指定集合是否为该方法参数集合的子集。

  • pop

随机移除元素

  • remove

移除指定元素

  • symmetric_difference

返回两个集合中不重复的元素集合。

  • symmetric_difference_update

移除当前集合中在另外一个指定集合相同的元素，并将另外一个指定集合中不同的元素插入到当前集合中。

  • union

返回两个集合的并集

  • update

给集合添加元素

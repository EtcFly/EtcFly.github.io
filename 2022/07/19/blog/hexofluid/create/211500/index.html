工作准备

注册GitHub账号

由于博客是基于GitHub Page托管的, 所以需要一个Github账号, 可通过 官网 进行注册。

安装git

安装git工具, git 工具用于上传博客到 GitHub 平台, 将 hexo 部署到GitHub上。安装 Git比较简单, 直接下载进行安装即可。官网

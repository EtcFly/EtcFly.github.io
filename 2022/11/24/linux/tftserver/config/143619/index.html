tftp服务器配置

安装服务端

安装

sudo apt install tftp-server

安装完成后执行:
netstat -a | grep -i tftp

显示如下:

表示服务端启动完成。

配置服务端根目录

  默认tftp服务端根目录在/var/lib/tftboot目录, 有时候非root用户可能没有权限, 切换服务根目录。配置文件在/etc/default/tftp-hpa

  • 重启服务
    service tftpd-hpa restart

客户端安装

安装

sudo apt install tftp

安装完成即可

使用

  假设已经启动一台服务器, ip:192.168.125.29, 则

  • 建立连接

tftp 192.168.125.29

  • 获取文件
  • 上传文件
